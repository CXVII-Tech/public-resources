# Visual Studio Code Cheatsheet

## Multi-line editing keyboard shortcuts

**Edit multiple lines**
- `Option+Click` (Mac) / `Alt+Click` (Windows/Linux) - Add cursor at each click position
- `Cmd+Option+↑/↓` (Mac) / `Ctrl+Alt+↑/↓` (Windows/Linux) - Add cursor to line above/below
- `Cmd+Option+Shift+↑/↓` (Mac) / `Ctrl+Alt+Shift+↑/↓` (Windows/Linux) - Add cursor to all lines above/below
- `Option+Shift+I` (Mac) / `Alt+Shift+I` (Windows/Linux) - Add cursor to end of each selected line

**Column/box selection**
- `Option+Shift+Mouse Drag` (Mac) / `Alt+Shift+Mouse Drag` (Windows/Linux) - Column selection
- `Cmd+Option+Shift+→/←` (Mac) / `Ctrl+Alt+Shift+→/←` (Windows/Linux) - Expand/reduce column selection

**Select multiple occurrences**
- `Cmd+D` (Mac) / `Ctrl+D` (Windows/Linux) - Select a word or piece of text
- `Cmd+D` (Mac) / `Ctrl+D` (Windows/Linux) - Select next occurrence of that word or piece of text (repeat to expand selection)
- `Cmd+Shift+L` (Mac) / `Ctrl+Shift+L` (Windows/Linux) - Select all occurrences of current selection

**Quick tips**
- `Esc` - Exit multi-cursor mode
